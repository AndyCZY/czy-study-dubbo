## Dubbo 简介
Dubbo是一个分布式服务框架，是阿里巴巴的开源项目，被国内电商及互联网项目中所使用。         
Dubbo 致力于提供高性能和透明化的RPC远程服务调用方案，以及SOA服务治理方案。简单的说，dubbo就是个服务框架，       
如果没有分布式的需求，其实是不需要用的，只有在分布式的时候，才有dubbo这样的分布式服务框架的需求，并且本质上            
是个服务调用的东西，说白了就是个远程服务调用的分布式框架。
